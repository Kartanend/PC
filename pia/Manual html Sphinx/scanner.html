scanner module¶

scanner.arp_scan(ip)[source]¶

Performs a network scan by sending ARP requests to an IP address or a range of IP addresses.

Parameters

ip (str) – An IP address or IP address range to scan. For example: - 192.168.1.1 to scan a single IP address - 192.168.1.1/24 to scan a range of IP addresses.
